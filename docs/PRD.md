# Product Requirements Document (PRD)
**Product Name:** ResumeAI (Arcade Edition)
**Phase:** MVP (Minimum Viable Product)

## 1. Executive Summary
ResumeAI is a gamified, brutal, and highly actionable resume improvement tool. It transitions the standard "ATS checker" from a boring corporate utility into an engaging, interactive experience for Gen-Z and millennial developers. It not only roasts the user's current resume but features an AI "Chameleon Architect" to instantly rebuild the resume into industry-standard formats.

## 2. Target Audience
* Junior to Mid-level Software Engineers.
* Recent college graduates (like B.E. students preparing for placements).
* Job seekers experiencing high rejection rates.

## 3. Core Features
* **Gamified ATS Score:** A visual "Health Bar" representing resume survival chances.
* **The Brutal Roast:** A 2-sentence savage, humorous critique of the uploaded resume.
* **Missing Gear (Skills Gap):** Extraction and display of missing critical industry keywords.
* **Quest Log (Actionable Advice):** Strictly formatted, parsed bullet points offering concrete improvement steps.
* **Chameleon Resume Architect:** An AI agent that rewrites and reformats the user's resume into one of three target styles (Standard ATS, Startup Modern, Executive) and outputs renderable HTML for immediate PDF printing.

## 4. User Flow
1. User lands on the dark-themed, arcade-style web interface.
2. User drops a `.pdf` resume into the upload zone.
3. User clicks "ROAST ME ??".
4. The system parses the PDF, calculates a score, and displays the Roast, Skills, and Quest Log.
5. User selects a target company style from a dropdown and clicks "BUILD IT ???".
6. The system generates a clean HTML resume.
7. User clicks "Print to PDF" to save the rebuilt resume.
